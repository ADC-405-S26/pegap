# A dataset to be used to practice with the porfolio analysis functions.

A dataset to be used to practice with the porfolio analysis functions.

## Usage

``` r
code_script
```

## Format

### `code_script`

A data frame with 5 rows and 5 columns:

- fund:

  Name of the fund (character)

- vintage_year:

  Year the fund began investing (integer)

- commitment:

  Total capital committed in USD (numeric)

- called:

  Capital called to date in USD (numeric)

- recallable:

  Recallable distributions in USD (numeric)
